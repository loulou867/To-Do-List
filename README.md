# To-Do-List
A simple To-Do-List made by myself because I was bored, hope you enjoy it


Requirement : 
tkinter

Installations : 

Open a Terminal
Windows :
- "pip install tkinter"
Linux : 
- "sudo python3 -m pip install tkinter"

How to use it :
Launch the script by the Terminal 
Type : "cd /directory/to/the/script/"
Windows : 
- "python ToDoList.py"
Linux
- "python3 ToDoList.py"
